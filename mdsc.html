Mars Directive: Sentient Convergence

Mars Directive: Sentient Covnergence is a third-person sci-fi action adventure shooter video game, set in the near feature that is set on Mars. This project includes the different documents typical of the life cycle of a video game and a working prototype that includes the main Game Loop, Win and Lose Conditions of the concept, a Tutorial Level and a Level Blockout consistent with the narrative of the game. The game implements traversal mechanics, interactable items and a shooting system. These provide the basic experience expected in the game. The concept has been carefully designed according to the common design steps typical of Video Games.
